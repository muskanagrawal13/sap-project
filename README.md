# 📊 SAP Record-to-Report (R2R) Capstone Project

## 📌 Overview
This project demonstrates the **Record-to-Report (R2R)** process in SAP Financial Accounting (FI). It covers the complete financial lifecycle from **journal entry posting to financial reporting and period closing**.

The project combines **functional SAP concepts**, **technical implementation**, and a **mini dashboard** to simulate real-time financial insights.

---

## 🎯 Objectives
- Understand the end-to-end R2R process  
- Implement SAP FI transactions (T-Codes)  
- Simulate financial data and reporting  
- Build a simple dashboard for visualization  
- Prepare industry-level documentation  

---

## 🔄 R2R Process Flow
1. Journal Entry Posting (FB50)  
2. General Ledger Update (FS10N)  
3. Account Reconciliation (FBL3N)  
4. Adjustments & Accruals (FBS1)  
5. Financial Statement Preparation (F.01)  
6. Period Closing (OB52)
